# Benihbibit Web

BenihBibit is a modern e-commerce web application for browsing and purchasing plant seeds and seedlings.

## Features

- Product catalog (browse seeds & plants)
- Search & filter products
- Product detail page
- Shopping cart management
- Checkout flow (basic / mock)
- Responsive design (mobile-first)
- Fast performance & optimized UI

## Tech Stack

- React
- Tailwind CSS
- REST API (Benihbibit API)
- React Router
- Netlify

## Getting Started

### Installation

```sh
git clone https://github.com/bando9/benihbibit-web.git
cd benihbibit-web
npm install
# or
bun install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

or

```bash
bun dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```
